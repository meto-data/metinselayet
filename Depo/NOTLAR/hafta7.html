Ders Notları: CSS'e Giriş ve Biçimlendirme

1. CSS Nedir ve Seçiciler (Selectors)

Sözdizimi: Seçici { özellik: değer; } (Noktalı virgülü unutma!)
  • Element Seçici : Sadece etiket adını yazarız. (Örn: p { color: red; }) Tüm P'leri etkiler.
  • Class (.) : Sınıf seçicidir. Başına nokta konur. (Örn: .kirmizi-yazi) Birden fazla elemana verilebilir.
  • ID (#) : Kimlik seçicidir. Başına diyez konur. (Örn: #ozel-kutu) Sayfada tek olmalıdır.
  • Evrensel (*) : Sayfadaki HER ŞEYİ seçer.

🔥 Canlı Test: CSS Öncelik Sırası (Basamaklanma)

Aşağıdaki metne CSS'te hem Class (mavi) hem de ID (kırmızı) rengi verdik. Ekranda hangi renk görünüyor? Kırmızı! Çünkü ID, Class'ı ezer.

Benim ID'm kırmızı, Sınıfım mavi diyor. Kimin sözü geçecek? Tabii ki ID'nin!

Ama dikkat: Eğer bu elemana style="color: purple;" verseydik (Inline CSS), hepsini ezip Mor olurdu. Çünkü Inline CSS en baskınıdır!

2. Renk Tanımlama Formatları

CSS'te renkleri 4 farklı şekilde verebiliriz:

tomato
#3498db
rgb(46,204..)
rgba(... 0.5)

  • Siyah: #000000 veya rgb(0,0,0)
  • Beyaz: #FFFFFF veya rgb(255,255,255)
  • Saydamlık: RGBA sonundaki "A" harfi (Alpha) 0 ile 1 arası değer alır. 0.5 yarı saydam demektir. (En sağdaki kutu).

3. Çerçeveler (Borders)

Çerçeve (border) elemanın dışına çekilir. İçindeki metni ezmez.

border: 4px solid red; (Düz Çizgi)
border: 4px dashed green; (Kesik Kesik)
border: 4px dotted yellow; (Noktalı)
border: 6px double blue; (Çift Çizgi)
border-radius: 20px; (Köşeleri Yuvarlatma)

4. Dış Boşluk (Margin) ve Çakışma (Collapse)

Margin, elemanın çerçevesinin dışından diğer elemanlara uyguladığı itme gücüdür.
margin: auto; kullanırsak, eleman sağdan ve soldan eşit iteceği için tam ortaya hizalanır.

🔥 Canlı Test: Margin Collapse (Çakışma)

Aşağıdaki sarı kutunun alt margin'i 50px. Mavi kutunun ise üst margin'i 50px. Normalde aralarında 100px boşluk olması lazım değil mi? Hayır! Çakışırlar (Collapse) ve aradaki boşluk sadece 50px olur (Büyük olanın sözü geçer).

Benim alt boşluğum 50px (margin-bottom: 50px)

Benim üst boşluğum 50px (margin-top: 50px)

🔥 Derse Aşkın: Sınav Odaklı Vurgular

  • Banka Soru (Öncelik Sırası): Bir elemana hem etiket, hem class, hem de ID ile farklı renkler verilirse kim kazanır?
    Cevap: ID kazanır! Sıralama: Inline > ID > Class > Element şeklindedir.
  • Şaşırtmaca (Margin Collapse): Üstteki div'in margin-bottom: 40px, alttaki div'in margin-top: 30px ise aralarındaki toplam mesafe nedir?
    Cevap: Toplamları olan 70px DEĞİL, büyük olan 40px'tir. Buna margin çakışması denir.
  • Renk Kodları (Test Sorusu):
    Beyaz: #FFFFFF (veya rgb(255,255,255))
    Siyah: #000000 (veya rgb(0,0,0))
    Kırmızı: #FF0000 (veya rgb(255,0,0))
  • Margin Nereye Boşluk Verir?
    Cevap: Border'ın (Çerçevenin) DIŞINA.
